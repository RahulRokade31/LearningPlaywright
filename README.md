# LearningPlaywright

Simple sample project for learning Playwright and Node.js basics.

## Contents
- `01_basic.js` — simple console log
- `02_step_by_step.js` — (example file)
- `03_verify_setup.js` — prints `process.platform`
- `04_hot_code.js` — small hot-code loop demo

## Run
From the `chapter_01_Basics` folder run:

```powershell
node 01_basic.js
node 03_verify_setup.js
node 04_hot_code.js
```
